1. Comments

Comments in C are used to add explanatory or descriptive text within the source code. They are ignored by the compiler and have no effect on the execution of the program.

Syntax:

2. Variables

Variables in C are used to store and manipulate data. They have a specific data type, a name, and a value. Before using a variable, it needs to be declared, specifying its data type.

Syntax: 

data_type variable_name;

Example: 


int age;        // Declaration of an integer variable named "age"
age = 25;       // Assignment of the value 25 to the variable "age"

3. Data Types

Data types in C specify the type and size of data that a variable can store. They define the operations that can be performed on the variable.

1. int: Represents integer values, which can be positive, negative, or zero.

2. float: Represents single-precision floating-point n

3. double: Represents double-precision floating-point numbers with higher precision than float.

4. char: Represents a single character enclosed in single quotes.

5. _Bool: Represents Boolean values (true or false).

Syntax: 


int variable_name;
float variable_name;
double variable_name;
char variable_name;
_Bool variable_name;

Example: 


int num = 10;
float pi = 3.14;
double distance = 10.5;
char grade = 'A';
_Bool isPassed = 1;  // 1 represents true

5. printf() and scanf() Functions

The printf() function is employed to present formatted output on the console or terminal. It enables the display of values and text while providing options for precise formatting.

The scanf() function is used to read formatted input from the user via the keyboard. It allows you to accept input and assign it to variables based on the specified format.

Syntax: 


printf("format_string", arguments);
scanf("format_string", &variable1, &variable2, ...);

Example: 


#include <stdio.h>
int main(){
int age = 25;
printf("My age is %d years.\n", age); // My age is 25 years.
 
int num;
printf("Enter a number: ");   // Enter a number: 10
scanf("%d", &num);
printf("Number: %d",num);  // Number: 10
return 0;
}

6. Formatted Input and Output

In C, both printf() and scanf() functions support formatting options using format specifiers. Format specifiers are placeholders that define the type and format of the data being inputted or outputted.

Syntax: 


%d: Used for integers.
%f: Used for floating-point numbers.
%c: Used for characters.
%s: Used for strings.

Example: 


int num = 10;
float pi = 3.14;
char grade = 'A';

printf("Number: %d\n", num);
printf("Pi: %.2f\n", pi);
printf("Grade: %c\n", grade);

/* Output: 
Number: 10
Pi: 3.14
Grade: A
*/

7. Conditional Statements (if, else if, else)

The if statement allows you to execute a block of code based on a condition. If the condition evaluates to true, the code enclosed within the if block is executed.

Syntax: 


// if  
if (condition) {
     // Code block for execution
}
 
//  else if
if (condition1) {
     // Code block for execution
 }
 else if (condition2) {
     // Code block for execution
 }

// else
 if (condition) {
     // Code block for execution only when true
 }
 else {
     // Code block for execution only when false
 }

Example: 


// if
int age = 20;
if (age >= 18) {
     printf("You are an adult.");
}
 
// else if
int score = 85;
if (score >= 90) {
    printf("Excellent!");
}
else if (score >= 80) {
    printf("Good!");
}
else if (score >= 70) {
    printf("Average.");
}
else {
    printf("Keep working hard!");
}

// else
int num = 10;
if (num % 2 == 0) {
    printf("Even number.");
}
else {
    printf("Odd number.");
}

8. Loops (for, while, do-while)

The for loop allows you to repeatedly execute a block of code for a specified number of times. It consists of an initialization, a condition, and an increment/decrement expression.

The while loop repeatedly executes a block of code as long as the specified condition is true. It verifies the condition prior to each iteration.

The do-while loop is similar to the while loop but checks the condition after executing the block of code. It guarantees that the code is executed at least once.

Syntax: 


// for loop
for (initialization; condition; increment/decrement) {
     // each iteration code  
}

// while loop
while (condition) {
     // each iteration code  
}
 
// do while loop
do {
     // each iteration code  
} while (condition);

Example: 


// for
for (int a = 1; a <= 5; a++) {
     printf("%d ", a);
}
// Output: 1 2 3 4 5



// while
int i = 5;

while (i >= 0) {
    printf("%d ", i);
    i--;
}
// Output: 5 4 3 2 1 0


//do-while

int i = 1;
do {
    printf("%d ", i);
    i++;
} while (i <= 5);
// Output: 1 2 3 4 5

9. Switch Statement

The switch statement provides a convenient way to choose between multiple cases based on the value of an expression. It simplifies the code when dealing with multiple conditions.

Syntax: 


switch (expression) {
     case value1:
          //  if expression equals value1
          break;
     case value2:
          //   if expression equals value2
          break;
     // Additional cases...
     default:
          //  if expression doesn't match any case
}

Example: 


int choice = 2;
switch (choice) {
     case 1:
          printf("You selected Option 1.");
          break;
     case 2:
          printf("You selected Option 2.");
          break;
     case 3:
          printf("You selected Option 3.");
          break;
     default:
          printf("Invalid option.");
}
// Output: You selected Option 2.

10. Break and Continue statements

The break statement is employed to prematurely terminate a loop or switch statement. When encountered, it immediately ends the current iteration of the loop or entirely exits the switch statement.

The continue statement is used to skip the remaining code within a loop iteration and proceed to the next iteration. It allows you to bypass certain parts of the loop's code.

Syntax: 


break;

continue;

Example: 


// break
for (int i = 1; i <= 10; i++) {
     if (i == 5) {
          break;
     }
     printf("%d ", i);
}
// Output: 1 2 3 4  


// continue
for (int i = 1; i *lt;= 5; i++) {
     if (i == 3) {
         continue;
     }
     printf("%d ", i);
 }
// Output: 1 2 4 5

14. Structures

Structures in C are user-defined data types that allow you to group different variables with different data types under a single name. They provide a convenient way to represent complex data structures.

Syntax: 


struct structure_name {
     data_type member1;
     data_type member2;
     // Additional members...
};

Example: 


struct Person {
     char name[20];
     int age;
};
struct Person person1;
strcpy(person1.name, "Krishna");
person1.age = 19;

printf("Name: %s\n", person1.name);
printf("Age: %d\n", person1.age);
     
/* Output
Name: Krishna
Age: 19
*/

15. Unions

Unions in C are similar to structures as they allow you to group variables of different data types. However, unions can only store one value at a time, sharing the same memory space for all members. The size of a union is determined by its largest member.

Syntax: 


union union_name {
     data_type member1;
     data_type member2;
     // Additional members...
};

Example: 


union Data {
     int num;
     float value;
};

union Data data;
data.num = 10;
printf("Number: %d\n", data.num);

data.value = 3.14;
printf("Value: %.2f\n", data.value);

/*
Output: 
Number: 10
Value: 3.14
*/

Note: When a new value is assigned to one member of a union, the previous value is overwritten and becomes inaccessible.

16. File Handling

File handling in C allows you to read from and write to files on disk. It provides a way to store and retrieve data persistently. C provides standard library functions for file operations.

To perform file handling in C, you need to follow these steps:

1. Open the file using the fopen() function, which returns a file pointer.

Perform the desired read or write operations using functions like fread(), fwrite(), fprintf(), fscanf(), etc.

Close the file using the fclose() function to release system resources.

Example: 


// writing to a file

#include <stdio.h>

int main() {
     FILE *file = fopen("data.txt", "w");
     if (file == NULL) {
          printf("File could not be opened.");
          return 1;
     }

     fprintf(file, "Hello, World!");

     fclose(file);

     return 0;
}

17. Preprocessor Directives

Preprocessor directives in C are special instructions that are processed by the preprocessor before the compilation of the program. They begin with a hash symbol (#) and provide instructions for the preprocessor to modify or include certain sections of code. 


#include: Used to include header files.
#define: Used to define constants or macros.
#ifdef and #endif: Used for conditional compilation.
#pragma: Used for implementation-specific instructions.

Example: 


#include <stdio.h>

#define PI 3.14159

int main() {
     float radius = 5.0;
     float area = PI * radius * radius;

     printf("Area of the circle: %.2f", area);

     return 0;
}

18. Enumerations

Enumerations in C allow you to define a set of named constants. They provide a way to represent a group of related values with meaningful names, making the code more readable and maintainable.

Syntax: 


enum enum_name {
     value1,
     value2,
     // Additional values...
};

Example: 


#include <stdio.h>

enum Day {
     MON,
     TUE,
     WED,
     THU,
     FRI,
     SAT,
     SUN
};

int main() {
     enum Day today = WED;

     if (today == SAT || today == SUN) {
          printf("It's the weekend!");
     } else {
          printf("It's a weekday.");
     }

     return 0;
}

19. Recursion

Recursion in C refers to the process of a function calling itself directly or indirectly. It allows the solution of complex problems by breaking them down into smaller, simpler instances.

A recursive function consists of two parts:

1. Base Case: A condition that specifies when the recursion should stop.

2. Recursive Case: The part of the function where it calls itself to solve a smaller instance of the problem.

Example: 


#include <stdio.h>

int factorial(int n) {
     if (n == 0) {
          return 1; // factorial of 0 is 1
     } else {
          return n * factorial(n - 1); // Recursive case: n! = n * (n-1)!
     }
}

int main() {
     int num = 3;
     int result = factorial(num);

     printf("The Factorial of %d is %d", num, result);  

     return 0;
}

// Output:
// The Factorial of 3 is 6
