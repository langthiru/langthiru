1. Comments

2. Variables

Variables in R are containers used to store data values. They have names and can hold different types of data, such as numbers, strings, logical values, and more. Variables allow manipulation and retrieval of data during program execution.

Syntax: 


variable_name = data 
     (or)
variable_name <- data

Example: 


# Declaring and initializing an integer variable in R
my_variable <- 5

# Updating the value of the variable
my_variable <- 10

# Declaring and initializing a string variable in R
message <- "Hello, World!"

# Printing the value of the variable
print(my_variable)
print(message)
  
# Output: 
# [1] 10
# [1] "Hello, World!"

4. Operators

Operators in programming languages are symbols or keywords that perform specific operations on operands (variables, values, or expressions). They enable manipulating data, performing calculations, comparing values, and controlling program flow.

5. Vectors

In R, a vector is a one-dimensional data structure that can hold multiple values of the same data type. It is the most basic and commonly used data structure in R, allowing for efficient storage and manipulation of homogeneous data elements.

Example: 


# Creating a numeric vector using the c() function
numeric_vector <- c(1, 2, 3, 4, 5)

# Creating a character vector using direct assignment
character_vector <- c("apple", "banana", "orange")

# Accessing elements of a vector
print(numeric_vector[3])  # Output: 3
print(character_vector[2])  # Output: banana

6. Matrices

A matrix in R is a two-dimensional data structure consisting of rows and columns. It is useful for handling tabular data and performing matrix operations.

Example: 


# Creating a matrix
my_matrix <- matrix(c(1, 2, 3, 4, 5, 6), nrow = 2, ncol = 3)
# Accessing elements of a matrix
print(my_matrix[1, 2])  # Output: 2

# Performing matrix transpose
transposed_matrix <- t(my_matrix)
print(transposed_matrix)

7. Arrays

An array in R is a multi-dimensional data structure that can hold elements of the same data type. It can have more than two dimensions and is useful for handling complex data sets.

Example: 


# Creating a 3D array
my_array <- array(c(1, 2, 3, 4, 5, 6), dim = c(2, 3, 2))
# Accessing elements of an array
print(my_array[1, 2, 1])  # Output: 2

# Performing operations on arrays
summed_array <- my_array + 2
print(summed_array)

8. Lists

In R, a list is a versatile data structure that can contain elements of different data types, such as vectors, matrices, data frames, and even other lists. It allows for flexible storage and manipulation of heterogeneous data.

Syntax: 


# Creating a list
my_list <- list(element1, element2, element3, ...)

Example: 


# Creating a list with different data types
my_list <- list("apple", c(1, 2, 3), my_matrix, TRUE)

# Accessing elements of a list
print(my_list[[1]])  # Output: apple
print(my_list[[2]])  # Output: 1 2 3

9. Data Frames

A data frame in R is a two-dimensional tabular data structure similar to a table in a relational database. It consists of rows and columns, where each column can have a different data type. Data frames are commonly used for data analysis and manipulation.

Syntax: 


# Creating a data frame
my_data_frame <- data.frame(column1, column2, column3, ...)

Example: 


# Creating a data frame
my_data_frame <- data.frame(name = c("John", "Emily", "Michael"),
                              age = c(25, 30, 28),
                              salary = c(50000, 60000, 55000))

# Accessing elements of a data frame
print(my_data_frame$name)    # Output: John Emily Michael
print(my_data_frame[2, 3])   # Output: 60000

10. Subsetting and Indexing

Subsetting and indexing in R refer to the process of extracting specific elements or subsets of data from vectors, matrices, lists, or data frames. It allows for selective data manipulation and analysis.

Syntax: 


# Subsetting and indexing
subset <- object[index]

Example: 


# Subsetting a vector
subset_vector <- numeric_vector[3:5]

# Indexing a matrix
indexed_matrix <- my_matrix[2, ]

# Subsetting a data frame
subset_data_frame <- my_data_frame[my_data_frame$age > 25, ]

sorted_data <- order(data$column)
sorted_data <- data[order(data$column), ]

filtered_data <- my_data_frame[my_data_frame$age > 25, ]

sorted_data <- my_data_frame[order(my_data_frame$age), ]

13. Aggregation and Grouping

Aggregation refers to summarizing data based on specific criteria, while grouping involves dividing data into subsets based on categorical variables.

Syntax: 


# Aggregation
aggregated_data <- aggregate(values ~ grouping_variable, data = my_data_frame, FUN = function_name)

# Grouping
grouped_data <- group_by(my_data_frame, grouping_variable)
summarised_data <- summarise(grouped_data, summary_statistic = function_name(values))

Example: 


# Aggregating data based on a grouping variable
aggregated_data <- aggregate(salary ~ department, data = my_data_frame, FUN = mean)

# Grouping data by a variable and calculating summary statistics
grouped_data <- group_by(my_data_frame, department)
summarised_data <- summarise(grouped_data, average_salary = mean(salary))

14. if

Syntax: 


if (condition) {
     # Code block  
}

Example: 


a <- 10

if (a > 0) {
     print("a is positive number")
}

15. else

The "else" statement is used in conjunction with the "if" statement to provide an alternative code block to be executed when the specified condition evaluates to false.

Syntax: 


if (condition) {
     # Code block  
} else {
     # Code block 
}

Example: 


a <- -5

if (a > 0) {
     print("a is positive")
} else {
     print("a is non-positive")
}

# [1] "a is non-positive"

16. elif

In R, the "elif" statement does not exist. Instead, you can use nested "if" and "else" statements to handle multiple conditions. Each nested "if" statement can have its own condition to evaluate.

Syntax: 


if (condition1) {
     # Code block  
} else if (condition2) {
     # Code block  
} else {
     # Code block  
}

Example: 


a <- 0

if (a > 0) {
     print("x is positive")
} else if (a < 0) {
     print("a is negative")
} else {
     print("a is zero")
}
    
# [1] "a is zero"

17. for

Syntax: 


for (variable in sequence) {
     # Code block  
}

Example: 


for (i in 1:5) {
     print(i)

18. while

A "while" loop is a control flow statement used to repeatedly execute a block of code as long as a given condition remains true. It allows for continuous iteration until the condition becomes false.

Syntax:

19. repeat

A "repeat" loop is a control flow statement used to execute a block of code indefinitely until a specific condition is met. It provides a way to create an infinite loop that must be manually terminated using the "break" statement.

Syntax: 


repeat {
     # Code block  
     if (condition) {
          break  #To terminate the loop
     }
}

Example: 


i <- 1

repeat {
     print(i)
     i <- i + 1

     if (i > 5) {
     break
     }
}

# [1] 1
# [1] 2
# [1] 3
# [1] 4
# [1] 5

20. break and next

he "break" and "next" statements are control flow statements used within loops to alter the flow of execution. "break" is used to exit the current loop completely, while "next" is used to skip the current iteration and move to the next iteration.

Syntax: 


# break statement
for (variable in sequence) {
     if (condition) {
     break  # Exit the loop
     }
}

# next statement
for (variable in sequence) {
     if (condition) {
     next  # Skip the current iteration and move to the next
     }
}

Example: 


print("break statement")
for (i in 1:10) {
     if (i == 5) {
          break  # Exit the loop when i is 5
     }
     print(i)
}
print("next statement: ")
for (i in 1:10) {
     if (i %% 2 == 0) {
          next  # Skip even numbers
     }
     print(i)
}
  
# [1] "break statement"
# [1] 1
# [1] 2
# [1] 3
# [1] 4
# [1] "next statement: "
# [1] 1
# [1] 3
# [1] 5
# [1] 7
# [1] 9

25. ggplot2 Package

ggplot2 is a popular data visualization package in R that provides an implementation of the Grammar of Graphics. It offers a powerful and flexible system for creating high-quality and customizable plots. ggplot2 uses a layered approach to construct plots, allowing for easy customization and complex visualizations.

Example: 


library(ggplot2)

# Scatter plot
df <- data.frame(x = c(1, 2, 3, 4, 5), y = c(2, 4, 6, 8, 10))
ggplot(df, aes(x, y)) + geom_point() + ggtitle("Scatter Plot") +
     xlab("X") + ylab("Y")

# Bar plot
df <- data.frame(category = c("A", "B", "C", "D", "E"), value = c(10, 20, 15, 25, 30))
ggplot(df, aes(category, value)) + geom_bar(stat = "identity") +
     ggtitle("Bar Plot") + xlab("Category") + ylab("Value")

# Histogram
df <- data.frame(values = rnorm(100))
ggplot(df, aes(values)) + geom_histogram() +
     ggtitle("Histogram") + xlab("Values") + ylab("Frequency")

26. Plotly Package

Plotly is an interactive and web-based data visualization package in R. It allows you to create interactive and dynamic plots that can be easily shared and embedded in web applications or presentations. Plotly supports various types of plots, including scatter plots, line plots, bar plots, heatmaps, and more.

Example: 


library(plotly)

# Scatter plot
df <- data.frame(x = c(1, 2, 3, 4, 5), y = c(2, 4, 6, 8, 10))
plot_ly(df, x = ~x, y = ~y, type = "scatter", mode = "markers") %>%
     layout(title = "Scatter Plot", xaxis = list(title = "X"), yaxis = list(title = "Y"))

# Bar plot
df <- data.frame(category = c("A", "B", "C", "D", "E"), value = c(10, 20, 15, 25, 30))
plot_ly(df, x = ~category, y = ~value, type = "bar") %>%
     layout(title = "Bar Plot", xaxis = list(title = "Category"), yaxis = list(title = "Value"))

# Histogram
df <- data.frame(values = rnorm(100))
plot_ly(df, x = ~values, type = "histogram") %>%
     layout(title = "Histogram", xaxis = list(title = "Values"), yaxis = list(title = "Frequency"))

29. Hypothesis Testing

Hypothesis testing is a statistical technique used to make inferences or draw conclusions about a population based on sample data. It involves formulating a null hypothesis (H0) and an alternative hypothesis (HA), collecting and analyzing the data, and determining the statistical significance to accept or reject the null hypothesis.

Example: 


# Create two numeric vectors
group1 <- c(10, 15, 12, 18, 20)
group2 <- c(13, 16, 11, 19, 17)

# Perform an independent t-test
t_test <- t.test(group1, group2)

30. Regression Analysis

Regression analysis is a statistical technique used to model the relationship between a dependent variable and one or more independent variables. It aims to understand how the independent variables impact the dependent variable and make predictions or infer relationships based on the obtained regression model.

Example: 


# Create a data frame with variables
df <- data.frame(x = c(1, 2, 3, 4, 5), y = c(2, 4, 6, 8, 10))

# Fit a linear regression model
model <- lm(y ~ x, data = df)

# Print the regression model summary
summary(model)

31. ANOVA

ANOVA is a statistical technique used to compare the means of two or more groups in order to determine if there are significant differences between them. It assesses the variation between group means and within-group variation to determine if the observed differences are due to random chance or if they are statistically significant.

Example: 


# Create data vectors for each group
group1 <- c(10, 15, 12, 18, 20)
group2 <- c(13, 16, 11, 19, 17)
group3 <- c(8, 9, 14, 11, 15)

# Perform one-way ANOVA
anova_result <- aov(c(group1, group2, group3) ~ c(rep("Group 1", 5), rep("Group 2", 5), rep("Group 3", 5)))

# Print the ANOVA results
summary(anova_result)

32. Installing Packages

Syntax:

Example:
